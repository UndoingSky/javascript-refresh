# JavaScript Refresh

My journey to rebuild modern JavaScript fundamentals before moving to TypeScript, React, Next.js, APIs, databases, deployment, and Google Cloud.

## Goals

- Master modern JavaScript fundamentals
- Learn TypeScript
- Learn React
- Learn Next.js
- Build full-stack projects
- Deploy to Google Cloud

Started: June 2026

## Study Plan

- Keep each session to 30-60 minutes
- Use short explanations, a few examples, and one small exercise
- Focus on the JS topics that matter most for React first

## Roadmap

1. Core foundations: variables, types, scope, functions, arrow functions, objects, arrays
2. React-ready syntax: array methods, destructuring, spread/rest, modules
3. Async JavaScript: promises, async and await, error handling, fetch
4. Browser basics: DOM, events, event listeners
5. Transition topics: this, binding, class basics
6. Move to TypeScript, React, and Next.js
